Feedback

Suggestions for MiniMax

1. Add image upload for subject_reference

Allow purpose="subject_reference" in /v1/files/upload API for S2V-01 video generation, avoiding need for external image hosting.

2. Consistent parameter types in documentation

Clarify that file_id in voice_clone must be integer. Add language-specific type hints (TypeScript, Python) to code examples.

3. More descriptive error messages

Error 2013 "invalid params" should specify which parameter is invalid and expected format/type.

4. Document GroupId requirement prominently

GroupId is required for voice_clone and TTS but easy to miss. Consider making it a header instead of query parameter.
